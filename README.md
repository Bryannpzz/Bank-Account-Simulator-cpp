# Bank Account Simulator with c++

This is a simple console based bank account simulator written in C++.  

The goal of this project was to practice object oriented programming concepts such as:

- Classes and objects
- Encapsulation
- Private vs public members
- Menu driven programs
- Input validation
- Basic financial logic

## Features

- Create an account with an owner name and starting balance
- Deposit money
- Withdraw money (with insufficient funds protection)
- View account information
- Interactive console menu

## Concepts Practiced

This project helped reinforce:

- How objects manage their own state
- How to protect data using private variables
- How to structure a clean menu loop
- Basic real-world simulation logic
